# 🏢 Enterprise Process Manager (EPM)

[![Language](https://shields.io)](https://microsoft.com)
[![Database](https://shields.io)](https://microsoft.com)
[![Platform](https://shields.io)](https://microsoft.com)

An enterprise-grade application built on **C#** and **SQL Server** designed for modeling, managing, and optimizing business processes using UML diagrams. The system features an automated, high-performance operation logging mechanism for complete system auditability and compliance.

---

## 🎯 Key Features

*   **📊 UML Process Modeling** – Visual design, editing, and mapping of complex enterprise workflows.
*   **⚡ Operational Management** – Real-time monitoring, orchestration, and tracking of business process execution.
*   **📝 Automated Logging** – Secure, tamper-proof recording of all user actions, data mutations, and system events.
*   **🔒 Audit & Compliance** – Streamlined access to operation history, making security audits and compliance checks effortless.

---

## 👥 User Role Differentiation

The system implements a strict Role-Based Access Control (RBAC) model:

| Role | Permissions & Responsibilities |
| :--- | :--- |
| **👑 Administrator** | Full system access, user management, advanced permission configuration, and system audit log inspection. |
| **🛡️ Moderator** | Managing and approving UML process models, task execution control, and operational log review. |
| **👤 Employee** | Accessing assigned UML charts, executing daily operational tasks, and creating standard data entries. |

---

## 🛠️ Technology Stack

*   **Backend & Logic:** `C# (.NET)` — Ensures high performance, type safety, and robust business logic execution.
*   **Database & Storage:** `Microsoft SQL Server` — Provides ACID-compliant secure data storage, relational integrity, and fast, optimized transaction logging.
